Detailed Description

Gerencia a exibição da mensagem de Game Over, scores, botões de ação e a interação do jogador. _currentScore: pontuação na partida _highScore: maior pontuação (recorde) ALLEGRO_FONT* _font: fonte dos textos da tela _playAgainButton: botão jogar novamente _returnToMenuButton: botão para retornar para a tela inicial _exitGameButton: botão para sair do jogo.
